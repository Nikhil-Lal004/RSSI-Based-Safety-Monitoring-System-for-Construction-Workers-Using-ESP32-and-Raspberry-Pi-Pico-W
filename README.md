# RSSI-Based-Safety-Monitoring-System-for-Construction-Workers-Using-ESP32-and-Raspberry-Pi-Pico-W
RSSI Based Safety Monitoring System for Construction Workers Using ESP32 and Raspberry Pi Pico W
•	Construction sites involve high risk due to heavy machinery and restricted zones.
•	This project proposes an RSSI (Received Signal Strength Indicator) based safety monitoring system.
•	ESP32 acts as a wireless beacon in danger zones.
•	Raspberry Pi Pico W acts as a wearable safety tag for workers.
•	When the worker approaches the danger zone, RSSI increases and triggers an alert buzzer.
•	The system provides a low-cost, real-time and indoor-friendly safety solution.
